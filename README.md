# Lost & Found NG

Nigerian lost-and-found recovery platform.

## Structure
- `server/` — Node.js/Express/MongoDB backend
- `client/` — React (Vite) frontend

## Local development
### Backend
```
cd server
npm install
npm start
```
### Frontend
```
cd client
npm install
npm run dev
```

## Deployment
- Backend: Railway (root directory: `server`)
- Frontend: Vercel (root directory: `client`)
